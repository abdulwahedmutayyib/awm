# Calculator App (Kivy + CI/CD)

A Python calculator application built using Kivy with CI/CD pipeline using GitHub Actions, Docker, pytest, flake8 and coverage.

## Features
- Basic arithmetic
- Scientific operations (sqrt, sin, cos, tan, power)
- Dockerized
- Fully tested
- CI pipeline

## Run locally

```bash
pip install -r requirements.txt
python calculator.py
